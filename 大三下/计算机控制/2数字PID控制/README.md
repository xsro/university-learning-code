# 实验二 数字 PID 控制

## 三、实验要求

1. 用 MATLAB 中的 simulink 工具箱实现该系统结构。
2. 根据自己所学的知识，确定 PID 控制器的参数，并记录该过程。
3. 改变 PID 控制器的参数，观察系统的阶跃响应曲线，记录相对应的性能指标（动态和稳态性能指标）。
4. 考虑改进的 PID 控制算法，观察系统的阶跃响应曲线，记录相对应的性能指标（动态和稳态性能指标）。
